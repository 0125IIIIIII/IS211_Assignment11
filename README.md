'ToDo List' Flask web application to manage a weekend to-do list. 
(This application requires Flask installed in your virtual environment) 

Users can view predefined tasks.
Users can add new items with task description - email contact - difficulty level (Low, Medium, High).
Users can replace the entire list with a comma-separated set of task(s).
Users can clear the list entirely.
Users input is validated with flash messages for valid email format, valid priority selection, and non-empty task field, or clear the entire list.

Installation : 
Copy the project files, navigate to the project directory, create and activate a virtual environment in Python, install Flask and run the app - python todoapp.py
Visit  http://127.0.0.1:5000/  in your browser to view the application.
